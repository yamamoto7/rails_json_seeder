

### How to start development
#### clone repo
```
$ git clone git@github.com:yamamoto7/rails_json_seeder.git
$ cd rails_json_seeder
```

#### add or edit code
```
├─ lib/
   ├─ rails_json_seeder.rb # entry point
   └─ rails_json_seeder/
      └─ seeder.rb # core functions
```

#### try features
```
$ cd test/dummy/
$ rails db:seed
```
