Json形式で定義されたシードデータを簡単に取り込むことができるgemです

## クイックスタート
### Install
https://rubygems.org/gems/rails_json_seeder

Gemfileに本gemを追加してください。
```
gem 'rails_json_seeder'
```

追加後以下のコマンドを実行してください。
```
$ bundle install
```
